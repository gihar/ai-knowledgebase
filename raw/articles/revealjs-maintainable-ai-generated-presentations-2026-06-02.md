# Пересланный пост: Reveal.js для поддерживаемых AI-сгенерированных презентаций

Мне кажется, многие зря используют Claude Design для создания презентаций.

Если вам нужны слайды, которые потом можно нормально поддерживать и дорабатывать, попросите Claude генерировать презентацию на Reveal.js 

То есть обычные HTML- и Markdown-файлы.

Получаете сразу несколько плюсов:

→ исходники в виде текста
→ хранение в Git
→ удобное редактирование в любом редакторе кода
→ полный контроль над стилями и анимациями
→ возможность публиковать презентацию как обычный сайт

Самое интересное, что презентация перестаёт быть отдельным артефактом, который живёт своей жизнью где-то в PowerPoint или очередном AI-инструменте.

Она становится частью проекта.

Claude отлично справляется с генерацией структуры, контента и оформления, а вы можете просматривать изменения через diff, использовать шаблоны, автоматизировать обновление слайдов и работать с ними так же, как с любым другим кодом.

Для разработчиков это ощущается естественнее. 🥴

100% опенсорс
