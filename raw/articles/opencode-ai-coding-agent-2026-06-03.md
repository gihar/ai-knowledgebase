# Что такое OpenCode?

OpenCode — это open-source AI coding agent: автономный помощник для разработки, похожий по роли на Claude Code / Codex CLI, но провайдер-независимый.

Проще говоря: это CLI/TUI-инструмент, которому можно дать задачу по коду — и он сам будет читать проект, править файлы, запускать команды, рефакторить, писать тесты или делать ревью PR.

Что важно:

- Open-source: проект открытый.
- Provider-agnostic: можно подключать разные модели/провайдеры, например Anthropic, OpenRouter и т.п.
- Работает из терминала:
  - интерактивно через TUI;
  - одноразово командой `opencode run "..."`.
- Подходит для автономной разработки:
  - «почини баг»;
  - «добавь фичу»;
  - «отрефактори модуль»;
  - «проверь PR»;
  - «добавь тесты».
- Можно запускать параллельно в разных workdir/worktree, если нужно распараллелить задачи.

Пример использования:

```bash
opencode run "Добавь retry logic для API-запросов и обнови тесты"
```

Или для ревью:

```bash
opencode pr 42
```

Установка обычно такая:

```bash
npm i -g opencode-ai@latest
```

или через Homebrew:

```bash
brew install anomalyco/tap/opencode
```

После этого нужно настроить авторизацию:

```bash
opencode auth login
```

Коротко: OpenCode — это «автономный AI-разработчик в терминале», которого можно подключить к разным LLM и использовать для задач по коду.
